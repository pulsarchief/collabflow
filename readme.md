# CollabFlow

**CollabFlow** is a real-time team collaboration and project management platform built using **Spring Boot**, **React**, **PostgreSQL**, **Redis**, and **WebSockets**. It allows teams to manage projects, assign tasks, track progress, and communicate efficiently â€” all in one place.

## Features

-   User authentication with JWT (Spring Security)
-   Project and task CRUD operations
-   Role-based access control (Admin, Manager, Developer)
-   Real-time task updates using WebSockets
-   PostgreSQL for persistent storage
-   Redis caching for performance
-   REST API backend built with Spring Boot
-   Modern frontend built with React + TailwindCSS
-   RabbitMQ for event-driven communication
-   Dockerized for local development
